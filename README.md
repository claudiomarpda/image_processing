# Image Processing

<h2>Tools</h2>
<p>
<b>Python 2.7</b> <br>
<b>OpenCV 2</b>
</p>

<p>Ubuntu install:</p>

```
$ sudo apt-get install python-opencv
```

Testing OpenCV:

```
$ python
>>> import cv2
>>> print(cv2.__version__)
```

<h2>Run</h2>

Run program:

```
python program1.py
```

Run test:

```
$ ./run_tests.sh
```

Note: If the script doesn't run, it may be necessary to give the following permission:<br>
`$ chmod +x run_tests.sh`
